ClassyStates

A third-party agnostic, extensible state-handler

For each state you need:

var myState = new ClassyState({
  render: myFunction,
  remove: anotherFunction
});

You must create at least one state with the option defaultState: true

After you've created all your states, you should use ClassyState.getState() (returns current state) and ClassyState.setState(myState) which will deactivate the current state and activate whatever state you've passed to it.

If you set the option revertToDefault to true and run ClassyState.setState(myState); while myState is active, then it will be deactivated and the default state will be activated.
